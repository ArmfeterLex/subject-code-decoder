# Subject Code Decoder

Модуль для консольных приложений на C#, преобразующий буквенный код школьного предмета в его полное название.

<img width="532" height="126" alt="Снимок экрана 2026-03-13 202910" src="https://github.com/user-attachments/assets/c0e39fe8-48f0-4900-b7d0-b17d1d0c28fc" />

## Функциональность

- Ввод буквенного кода предмета
- Преобразование в полное название:
  ф → Физика
  м → Математика
  и → История
  г → География
  б → Биология
- Обработка некорректного ввода

## Пример работы

Введите название предмета: ф, м, и, г, б
ф
Физика

Введите название предмета: ф, м, и, г, б
х
Некорректная буква предмета

## Куда можно встроить

- В электронные журналы
- В системы составления расписания
- В приложения для школьных учителей
- В дневники учеников

## Требования

- .NET Core / .NET Framework
- Консольное приложение

## Примечание

Набор предметов расширяется добавлением новых case.
